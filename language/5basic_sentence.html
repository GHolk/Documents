# 五大基本句子組合結構分析 #

所有語言中的句子，都是由不同的 **詞** ，
依五種基本結組合而成的。

 1. 主謂
 2. 偏正
 3. 述賓
 4. 述補
 5. 聯合


記得高中學過中文有四大句型，
和這個又有點不同，五種基本結構更靠近底層。

 - 敘述句：主語 + 述語 + 賓語。
 - 判斷句：主語 + 係詞 + 斷語。
 - 有無句：主語 + 述語 + 賓語。
 - 表態句：主語 + 表語。


 1. ""我必須藉由課前預習和課後複習來防止語言學概論被當。""

	 - 主謂： _我_ + _必須藉由課前預習和課後複習來防止語言學概論被當_ 
	 - 偏正： _必須_ + _藉由課前預習和課後複習來防止語言學概論被當_ 
	 - 述補： _藉由課前預習和課後複習_ + _防止語言學概論被當_ 
	 - 述賓： _藉由_ + _課前預習和課後複習_
	 - 並列： _課前預習_ + 課後複習_ 
	 - 述賓： _防止_ + _語言學概論被當_
	 - 主謂： _語言學概論_ + _被當_

 
 2. ""我今天騎車騎得很快趕在圖書館關門前去借關於養生跟廚藝的書。""

 3. ""小紅帽提著麵包籃在去找奶奶的路遇到了可怕又飢餓的大野狼。""

 4. ""野心勃勃的中國和追求獨立的台灣有著一言難盡的關係。""

 5. ""忙碌又辛苦的把冰箱搬出去的張媽媽加倍努力的打掃家裡。""
