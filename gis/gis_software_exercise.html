使用 QGIS 與 ArcGIS 製做地圖

雖然老師教我們用 ArcGIS ，但我懶得去系上的電腦教室， 我的電腦又只有裝 QGIS ，所以就先用 QGIS 做做看。 QGIS 和 ArcGIS 都吃 shape 檔，所以格式沒問題； 結果只差在 QGIS 的專案檔格式和 ArcGIS 的不一樣。

地圖製作

目標
做出台南市東區的小學地圖。
設計理念
提供儘可能詳細的資訊， 我希望可以把各資料都丟上地圖，同時保持版面整潔不雜亂。
資料
台南市東區面積、台南市東區各里面積、 台南市東區道路線段、台南市東區小學點位。

  1. 根據一般人的習慣，地圖上的道路是有寬度的。 所以把路的寬度調粗。

  2. 一般地圖是綠色的，所以把整個東區弄成綠色。 而路可能是灰色或白色。

  3. 為了區別各里，為里加上細黑色邊界。 不和道路混淆，固把道路調成白色。

  4. 如果里的圖層疊到道路上，面積的部份會把路蓋住， 但調透明度邊界又會變透明看不到。

    最後用多邊形轉線段的功能， 把里的圖層轉成線段，再疊到最上層。

    後來想到可以把里的面積顏色選成透明色， 而不直接調整個圖層的透明度，邊界就不會變透明。

  5. 小學的圖片本來想用小學女生的那種帽子， 但找不到手繪圖片，都只有照片。 最後用一個女學生漫畫截下來的。

  6. 放圖例時因為路是白色的，如果背景也是白色的會看不到， 所以圖例的背景隨便選了個順眼的顏色。

  7. 其它就沒什麼，把字調到合適的大小， 比例尺和圖例、標題位置喬一喬就 OK 。

  8. 發現可以輸出成 PDF ，而且檔案明顯小 PNG 很多！ 大概是 PDF 內可以存向量式資料，而 PNG 只能存點陣式。 作業給的 shape 都是向量式，所以存成 PDF 比較小。
